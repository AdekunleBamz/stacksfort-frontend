# StacksFort Frontend

A Next.js frontend for StacksFort - a decentralized finance platform on Stacks.

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- @stacks/connect

## Getting Started

```bash
npm install
npm run dev
```

## Project Structure

```
├── app/           # Next.js app router
├── components/    # React components
├── lib/          # Utilities and types
├── hooks/       # Custom hooks
└── utils/       # Helper functions
```
